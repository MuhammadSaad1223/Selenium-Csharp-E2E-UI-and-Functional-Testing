# Selenium C# SauceLabs Demo Test:

Automation testing framework built using:
- Selenium WebDriver
- C#
- NUnit
- Page Object Model (POM)

## Features
- Login automation
- Cart functionality
- Checkout flow
- Assertions
- Explicit waits
- UI Testing
- E2E Testing
- Funcional Testing
- Data-driven testing(Valid/Invalid User) 

## Tech Stack
- C#
- Selenium
- NUnit(FrameWork)
- FirefoxDriver
